cheerio

cheerio 为服务器特别定制的，快速，灵活， 实施的jQuery核心实现。
可用通过如下命令进行安装：
npm install cheerio

Features
相似的语法： 包括了jQuery核心的子集。
闪电般的快： 工作在一个非常简单，一致的DOM模型之上。高效的完成解析， 操作， 呈送。（基础的端到端的基准测试显示 Cheerio 大约比 JSDOM 快八倍）。
巨灵活： cheerio 封装了兼容 htmlparser。cheerio 几乎能够解析任何的 HTML 和 XML document。
